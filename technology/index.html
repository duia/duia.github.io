前端技术

  • Bootstrap 简洁、直观、强悍的前端开发框架，让web开发更迅速、简单。
  • bootstrap3-dialog 简化bootstrap的弹框
  • Font Awesome 一套绝佳的图标字体库和CSS框架
  • Color Admin是一个优秀的后台管理模板
  • bootflat 一个开源的基于 Bootstrap 3.3.0 CSS 的Flat UI KIT 框架
  • Jquery UI基于Jquery的开源网页用户界面代码库
  • Jquery EasyUI 帮助web开发者更轻松的打造出功能丰富并且美观的UI界面
  • Layui 经典模块化前端框架 github
  • zTree是一个依靠 jQuery 实现的多功能 “树插件”。优异的性能、灵活的配置、多种功能的组合是 zTree 最大优点。
  • ueditor UEditor是由百度web前端研发部开发所见即所得富文本web编辑器
  • ckeditor 富文本web编辑器，支持丰富的插件
  • datatables是一款jquery表格插件。它是一个高度灵活的工具，可以将任何HTML表格添加高级的交互功能。
  • flatpickr轻量级，功能强大的javascript datetimepicker，没有依赖关系
